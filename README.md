# LUVIPRO Landing Page

Production-ready frontend starter for the official LUVIPRO landing page.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- React
- Framer Motion
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

This first version is frontend only. Backend, database, authentication, payments, OCR, QR generation, and real map integrations are intentionally out of scope.
